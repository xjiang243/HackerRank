Procedure to set up Server:

1. Get into 'xampp' folder.
2. Right click 'xampp-control.exe', run as administrator.
3. In browser, enter 'localhost' to see if the server is up.
4. The root of the server 'htdocs' folder.
5. Create whatever a folder('phpdev' for example), put php project under it.
6. In browser, enter 'localhost/phpdev' to deploy the project.
